# 📝 Notion Import 가이드

이 폴더의 마크다운 파일들은 **Notion에 import**할 수 있도록 최적화된 형태입니다.

## 🚀 Notion에 한 번에 import하는 방법

### 방법 1: 전체 페이지 구조로 import (추천)

1. Notion에서 새 페이지를 생성합니다.
2. 상단의 `···` 메뉴 → `Import` 클릭.
3. `Markdown & CSV` 선택.
4. `notion/` 폴더의 파일들을 **전부 선택** 후 업로드.
5. 각 파일이 하위 페이지로 자동 구성됩니다.

### 방법 2: 한 페이지씩 복사-붙여넣기

각 `.md` 파일을 열고 전체 내용을 복사한 뒤, Notion 페이지에 붙여넣으면 됩니다.
(Notion은 마크다운 문법을 자동으로 변환합니다.)

## 📂 파일 구성

| 파일 | 주제 |
|------|------|
| `00_index.md` | 전체 목차 (루트 페이지) |
| `01_week3_process_thread.md` | 3주차 — 프로세스와 스레드 |
| `02_week4_cpu_scheduling.md` | 4주차 — CPU 스케줄링 |
| `03_week5_synchronization.md` | 5주차 — 프로세스 동기화 |
| `04_week6_deadlock.md` | 6주차 — 교착상태 |

## 💡 Notion에서 활용 팁

- **토글 블록 추가**: 긴 설명은 토글로 접어두면 깔끔합니다.
- **데이터베이스화**: "알고리즘 비교표" 등은 Notion 데이터베이스로 만들면 필터링/정렬이 편합니다.
- **체크리스트 활용**: 각 주차 README의 "학습 목표" 섹션을 체크박스로 전환하여 진도 관리에 사용하세요.
- **슬라이드 임베드**: `slides/` 폴더의 PDF를 Notion에 업로드하면 페이지 내 임베드로 볼 수 있습니다.
